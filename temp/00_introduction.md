# Fractal Network Documentation

## Introducción a Fractal Network

Fractal Network es una plataforma blockchain de próxima generación que utiliza una arquitectura de dos capas basada en un grafo acíclico dirigido (DAG). Esta innovadora estructura permite una ejecución eficiente y escalable de transacciones a través de un modelo impulsado por intents.

### Objetivo

El objetivo de Fractal Network es proporcionar un sistema descentralizado y offline-first que permita a los usuarios gestionar sus propios datos y transacciones de manera segura y eficiente. La plataforma está diseñada para ser modular, permitiendo la integración de diversas funcionalidades a través de componentes especializados.

### Arquitectura

Fractal Network se compone de dos capas principales:

1. **Capa de Agentes**: Esta capa se ejecuta en dispositivos de usuario final, como móviles y PCs. Los Agentes son responsables de generar intents firmados y mantener un micro-ledger local que refleja el estado del usuario.

2. **Capa de Custodios**: Compuesta por nodos descentralizados, los Custodios verifican los intents, gestionan el consenso y garantizan la disponibilidad de datos. Esta capa actúa como la base soberana de la red, asegurando la integridad y persistencia de las transacciones.

### Características Clave

- **Seguridad Criptográfica**: Fractal Network implementa primitivas criptográficas avanzadas, incluyendo pruebas de conocimiento cero (ZKP) y algoritmos de firma seguros, para proteger las transacciones y los datos de los usuarios.
  
- **Modelo Económico Dual**: La red utiliza dos tokens, $FRAC y $SOLV, para gobernanza y operaciones, respectivamente. Este modelo permite una gestión eficiente de los recursos y la incentivación de los participantes en la red.

- **Disponibilidad de Datos**: A través de técnicas como el pinning y la codificación de borrado, Fractal Network asegura que los datos de los usuarios estén siempre disponibles y protegidos contra pérdidas.

### Conclusión

Fractal Network representa un avance significativo en la tecnología blockchain, ofreciendo una solución robusta y flexible para la gestión de datos y transacciones. A medida que la red evoluciona, se espera que continúe innovando y adaptándose a las necesidades de sus usuarios.
